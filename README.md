# 💎 Calqly — Precision Financial Engineering

Calqly is a premium, high-performance financial calculator platform built for the modern wealth-builder. It provides sophisticated tools for tax optimization, debt management, and retirement planning with a focus on **Organic Clarity** and **Visual Excellence**.

![Calqly Preview](public/opengraph-image.png)

## 🚀 Key Features

- **🛡️ Smart Tax Engine**: Comprehensive Thai Personal Income Tax (PIT) calculator with support for SSF, RMF, Thai ESG, and the latest 2026 tax brackets.
- **🏠 Mortgage Strategist**: Advanced loan calculator with amortization schedules, step-up interest support, and "what-if" extra payment scenarios.
- **🌴 Retirement Blueprint**: Projection tools to calculate your "Magic Number" and safe withdrawal rates (SWR).
- **📈 ROI & Growth Analysis**: Compound interest and investment return calculators with real-time visualization.
- **📱 PWA & Offline-First**: Fully responsive design with local persistence (Zustand) for saving your financial projects securely.
- **🌐 Dual-Language**: Full support for English and Thai, optimized for local financial terminology.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Charts**: [Recharts](https://recharts.org/)
- **i18n**: [next-intl](https://next-intl-docs.vercel.app/)

## 📦 Getting Started

### Prerequisites

- Node.js 20.x or later
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/calqly.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## 🧪 Testing

We take financial accuracy seriously. Run our suite of stress tests to verify calculation logic across edge cases:

```bash
npx tsx run-tests.ts
```
